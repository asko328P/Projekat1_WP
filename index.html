My example projects:
Car fully modeled and rendered in Blender

Audi A4 fully modeled off of blueprints, here seen rendered with Eevee rendering engine. Extremely fast render time, more suited for animations as lighting will show artifacts.


Here it can be seen rendered with the Cycles rendering engine, which accomplishes this sophisticated lighting using pathtracing. Which means every 'photon' is simulated.

Special attention was paid to tiny details all around the car. As can be noticed by taking a look at all the nooks and crannies around the grille and headlight structure.


Here is the typical viewport setup. On the left side, nodes by which logic all materials get rendered which assemble the final frame. On the right, the camera through which the final picture will be viewed from.


A wireframe picture inside the working viewport which roughly shows the amount of polygons that make this model.


Home movie streaming site

This is a alternative to commercial movie streaming sites that uses self-hosted content to provide movies to every web-capable device in a household

Simple to use, this minimalistic design doesn't overwhelm the user with unnecessary information. Just the essentials that the user might need. The video player allows to enable or disable any downloaded subtitles

Maintaining the software is extremely user friendly to the user. With only 3 scripts that automatically download cover art, subtitles and general infos about the movie directly from IMDB

Every movie is comprised of at least 3 files: Cover art, information about the movie and the movie file itself. A optional subtitle file can also be found.
